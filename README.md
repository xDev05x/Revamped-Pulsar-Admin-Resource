# Pulsar Admin

This is a complete overhaul of the pulsar-admin resource from https://github.com/PulsarFW

It offers multiple new features like the following;
- Improved UI
- Unique Item Database Feature
- Admin Commands Panel with Usable Interaction
- Extended Character Information
- Intgration with my Custom Apartments Resource for Pulsar
- And more!


# Installation

This was pre-built, so should be a drag and drop. I always recommend keeping a backup of your original resource to be safe.
